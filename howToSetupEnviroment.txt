-> HOW TO SETUP ENVIRONMENT

	- Instalar o netbeans
	- Instalar o mysql
	- Descompactar o arquivo e importar o projeto para o Netbeans
	- mudar a Plataforma Java para "JDK 1.8 (Default)" em Propriedades do Projeto
	- adicionar as bibliotecas da pasta lib que foi disponibilizada (Projeto-				>Propriedades>Bibliotecas ->Adicionar JAR/Pasta)	
	- Criar o usuário 'editor'@'localhost' com senha 'editor' e demos todos os prilivégios à ele	
	(grant 	  all privileges to editor)	
	- Criar o banco 'srs'.
